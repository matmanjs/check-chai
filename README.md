# 单元测试代码生成工具

## 如何使用

首先安装依赖：

```bash
npm install
```

然后执行启动项目命令：

```bash
npm start
```

启动完成之后打开： http://localhost:3000/dashboard 即可。

## 如何部署到 GitHub Pages 

我们将样例部署到了 [GitHub Pages](https://pages.github.com/) ，地址为： https://matmanjs.github.io/code-generator-for-unit-testing/ 。

需要运行以下命令构建到 `docs` 文件夹下，并提交到 git 仓库中。

```bash
npm run deploy
```
