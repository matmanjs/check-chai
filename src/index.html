使用规范

chai.js 是一套TDD(测试驱动开发)/BDD(行为驱动开发)的断言库。
BDD，行为驱动开发（注重测试逻辑），TDD是测试驱动开发（注重输出结果）。

使用时需要先引入断言库：
var expect = require('chai').expect;

expect断言的写法都是一样的。
头部是expect方法，尾部是断言方法，比如equal、a/an、ok、match等。两者之间使用to或to.be连接
例如：
expect(4 + 5).to.be.equal(9); // 4加5应该等于9
